Letter-spacing: define o espaçamento entre letras/caracteres.

word-spacing: define o espaçamento entre palavras.

white-space: define como o espaço em branco se comporta.

word-wrap: se a palavra for muito grande com essa tag é definido se ocorre quebra ou toda na mesma linha. palavras chaves aceitas: normal, break-word.

word-break: define como ocorre a quebra de linha se a palavra for muito grande ou com hifen. palavras chaves aceitas: normal break-all - quebra todos os textos no limite definido. keep-all - dependendo do idioma esta chave se comporta como normal e em outras como break-all.

writing-mode: define a orientação do texto, se vertical ou horizontal. palavras chaves: horizontal-tb - horizontal top botom vertical-rl - vertical right left vertical-lr - vertical left right

text-overflow:

clip: se o texto for grande esse sera escondido

ellipsis: se o texto for grande esse sera escondido com "..."

string: se o texto for grande esse sera escondido pelo texto definido
