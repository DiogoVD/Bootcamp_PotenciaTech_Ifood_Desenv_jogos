Agrupamento de seletores

Podemos aplicar as mesmas regras CSS para varios seletores. Para isso, podemos separar os seletoresutilizando uma virgula(,)

Assim, não precisamos ficar copiando e colando os mesmos codigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
